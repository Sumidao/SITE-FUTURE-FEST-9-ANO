PROCESSO DO PROJETO.

Os nossos protótipos contêm o estilo e desenvolvimento, o projeto será em base no mundo atual mesmo tendo tecnologias avançadas e sabendo dos perigos mesmo assim não conseguem utilizar essas ferramentas pensando sustentavelmente sendo economicamente ou ecossistema mente. Então dessa forma esse projeto e pensado em criar um protótipo de garra robô que tem como função prestar assistência a quem precisa tornando mais fácil o dia a dia além de economizar o tempo, ou seja, um projeto que ajude a população.

O nosso grupo utilizou várias ferramentas relacionadas a tecnologia, como tinkercad onde colocamos o código para a garra mexer utilizando a linguagem de programação c, ele e introduzido no Arduino e só assim para o projeto fazer o movimento desejado, o projeto teve um tempo bom também para podermos personalizar, fazendo o desenho, pintando e detalhando cada parte de forma cuidadosa, além disso tudo colocamos o circuito no projeto com o motor para fazer o movimento no braço.

CONSTRUÇÃO DO PROJETO.

C+

Para o funcionamento de todo o circuito tivemos que trabalhar com a linguagem C+. Essa linguagem é normalmente utilizada em arduinos, para o funcionamento dos motores presentes no projeto.

MONTAGEM

Para construir o projeto, foi preciso utilizar a maquina a laser, para conseguir cortar a madeira com precisão, além de fazer a montagem e pintura de todos os componentes.

CIRCUITO

Para o projeto realmente conseguir fazer os movimentos desejados, fizemos um circuito com varios cabos soldados conectados ao arduino e desse jeito o projeto consegye funcionar
