PROJECT ENCHANTED.

Braço Robótico

Criando e Inovando, com uma forma única de forma sustentavel, pois atualmente, mesmo tendo tecnologias avançadas e sabendo dos perigos, não conseguem utilizar essas ferramentas pensando de forma sustentavel sendo economicamente ou de forma que priorize o ambiente e o ecossistema.

COM O QUE TRABALHAMOS?

C+

Para o funcionamento de todo o circuito tivemos que trabalhar com a linguagem C+. Essa linguagem é normalmente utilizada em arduinos, para o funcionamento dos motores presentes no projeto por exemplo.

MONTAGEM

Para construir o projeto, foi preciso utilizar a maquina a laser, para conseguir cortar a madeira com precisão, além de fazer a montagem e pintura de todos os componentes.

CIRCUITO

Para o projeto realmente conseguir fazer os movimentos desejados, fizemos um circuito com varios cabos soldados conectados ao arduino e desse jeito o projeto consegye funcionar

SOBRE O PROJECT ENCHANTED.

O nosso projeto foi feito em base de um braço robótico, um dispositivo mecânico composto por articulações atuadoras que imitam a estrutura de um braço humano. Essa estrutura é projetado para realizar tarefas automatizadas, como manipulação de objetos, montagem e muitas outras, e uma variedade de aplicações industriais e comerciais. Os braços robóticos são controlados por computadores e podem ser programados para executar um amplo local de movimentos precisos e repetitivos. Eles desempenham um papel fundamental em linhas de produção automatizadas e em diversas outras aplicações onde a precisão e a eficiência são essenciais.

Para conseguirmos mudar o mundo, precisamos de um novo jeito de pensar nele e no futuro ...
